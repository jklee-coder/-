# 独立管理后台使用说明

## 访问方式

管理后台现在已经独立出来，顾客无法在前台看到管理后台链接。

### 前台购物平台

- 访问地址：`http://localhost:3000/网上购物平台/购物平台.html`
- 功能：商品浏览、购物车、订单管理等顾客功能

### 后台管理系统

- 登录地址：`http://localhost:3000/网上购物平台/admin-login.html`
- 管理后台：`http://localhost:3000/网上购物平台/admin.html`

## 管理员账号

默认管理员账号：

- 邮箱：`admin@example.com`
- 密码：需要先在数据库中创建该用户

## 创建管理员账号

1. 首先在前台注册一个用户，邮箱使用 `admin@example.com`
2. 或者直接在数据库中插入管理员用户数据

## 安全特性

1. **权限分离**：顾客无法访问管理后台
2. **独立登录**：管理员需要单独登录
3. **权限验证**：所有管理 API 都需要管理员权限
4. **自动跳转**：未登录时自动跳转到登录页面

## 重启服务

修改后端代码后，需要重启 Node.js 服务器：

```bash
cd /d "e:\vscode\网上购物平台\backend"
npm start
```

## 功能模块

管理后台包含以下功能：

- 销售统计和数据分析
- 商品管理（添加、编辑、删除商品）
- 订单管理（查看、更新订单状态）
- 用户管理（查看用户信息）
